지금까지 배운 코딩(HTML) 2019.1.30 수요일

  1. 태그설명

    • 태그 : 내가 web상에서 표현하고자 하는 것을 구현시켜주는 기본 코딩문법
      <>라는 기호를 사용한다.
    • strong태그 : 글자를 진하게 한다.
    • u태그 : 밑줄
    • a태그 : 링크를 건다, (a href=""로 주소를 링크 건다.)(/a)
      • target="_blank"은 새창으로 링크를 연다.
      title=""을 쓰면 링크에 마우스를 올리면 내가 쓴 간단한 정보를 볼 수 있다.
    • h1~6태그 : 주로 제목에 쓴다. 중요도 순으로 글씨크기가 크다.
    • title태그 : 웹 창의 제목
    • br태그 : 줄을 바꾼다.
    • p태그 문단을 구분 짓는다.
    • img태그 : img src=""를 사용한다.
      width=""를 사용하여 사진의 크기를 조정한다.
    • 부모태그와 자식태그 :
      부모태그는 ol,ul을 쓴다. ol은 순서를 숫자로 표기, ul은 점으로 표기해준다. 자식태그는 li태그를 쓴다.
    • meta태그 : 파일을 어떠한 형식으로 읽으라는 태그 나는 주로 meta="uft-8"을 쓰게 될거 같다.

  2. 코딩의 기본 약속

    1. 가장처음에는 (!독타입 html)을 써준다.
    2. 그 다음에 전체를 (html태그)로 감싸준다.
    3. 본문을 설명해주는 내용들은 (head태그)로 감싸준다.
    4. 본문은 (body태그)로 감싸준다.

      5. 추신 : <>은 ()로 표현하였음.
